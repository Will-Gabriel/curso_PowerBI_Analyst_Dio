# ETL: Extract | Transform | Load

## Pipeline de Dados ETL
ETL é um processo essencial de integração de dados que coleta informações de várias fontes, limpa-as, padroniza-as e as carrega em um repositório centralizado, como um data warehouse ou data lake. Ele é fundamental para unificar dados legados, apoiar a análise de negócios, relatórios de inteligência e machine learning. 

### Etapas
1. `Extract`:
    * Extração dos Dados
    * `Clean` ou `Limpeza dos Dados`:
        * Fica entre a extração e transformação.
2. `Transform`:
    * Transformação dos Dados
3. `Load`:
    * Carregamento dos Dados
