# ELT: Extract | Load | Transform

## Pipeline de Dados ELT
ELT é uma abordagem moderna de integração de dados que migra dados brutos diretamente de fontes para um armazenamento de destino, como nuvem ou data lake, realizando a transformação posteriormente. 
* Solucionam o mesmo problema
* Diferença
    * Abordagem de transformação de Dados
    * Perspectiva sobre os Dados
    * Indenpendecia de TI
    * Profissionais de Analise de Dados

### Etapas
1. `Extract`:
    * Extração dos Dados
    * `Clean` ou `Limpeza dos Dados`:
        * Fica entre a extração e o carregamento.
2. `Load`:
    * Carregamento dos Dados
3. `Transform`:
    * Transformação dos Dados

## Processos de ELT
* `Extração`
    * Coleta de dados brutos
    * Fontes heterogêneas
    * Objetivos: Integração posterior
* `Carregamento`
    * Tranferência / Carregamento
    * Dados brutos
    * Data Warehouse ou Repositório Central
* `Transformação`
    * Transformação dos Dados dentro do repositório
    * Aplicação das Análises

## Vantagens do ELT
* Otimização de tempo
* Eficiência na implementação de projetos
* Menor dependências de TI
* Papel principal dos analistas
